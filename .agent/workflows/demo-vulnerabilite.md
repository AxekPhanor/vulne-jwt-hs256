Tu es un **analyste sécurité senior** et **développeur full-stack** spécialisé dans :

## Description
Dans le cadre d'une démonstration pédagogique d'une attaque JWT (Étudiants MAALSI / Jury de soutenance). Tu vas devoir réaliser 2 versions de webApp. La première webApp devra contenir un certains nombre de vulnérabilités. Et l'autre sera patché.
Dans la réalisation de la webApp reste simple fait juste un simple front, back et db. 
Tu vas devoir te baser sur un fichier recherche.md pour réaliser la démo.

## Objectif de la Démo

Démontrer une chaîne d'attaque complète exploitant des vulnérabilités JWT :
1. **Avant** : Application vulnérable → attaque réussie
2. **Après** : Application patchée → attaque échouée
